- 인터페이스 동아리원과 집부를 위한 앱입니다.

### 사용기술
#### MVVM
- 코드간 의존성 해결과 옵저빙을 위- 코드간 의존성 해결과 옵저빙을 위해 사용
#### LiveData
- 서버에서 받아온 데이터를 실시간으로 반영할 수 있게 하기위해 사용
#### DataBinding
- View - ViewModel 간 의존성 해결을 위해 사용
#### Retrofit
- 네트워크 통신을 쉽게 하기위해 사용
#### SingleTon
- repository 사용시 발생하는 메모리 낭비를 줄이기위해 사용
#### RxJava2
- 비동기 처리에 사용
